Agile

Agile model can be seen as Software Development Life Cycle model in which the focus is put on process adaptability and satisfied customer (Samuel, 2020: 19). The model has recently been widely used approach in the software development process (Samuel, 2020: 21). The five main values of using agile approach, is communication, simplicity, feedback, courage and humility. The approach can be characterized as a group of software development methodologies that are based on incremental and iterative approach.

Big Bang model

Big Bang model can be characterized as very simple, because it requires little planning, a lot of programming and funds. Very small amount of planning is used in this model. It is good model to use when customer is not sure what he or she wants and what will be necessary in the future. Big Bang model is not good for big software projects, and should better be used in the learning and experimenting process (Tutorialspoint, 2020b: 4).
